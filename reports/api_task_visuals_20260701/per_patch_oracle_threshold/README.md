## 结论

- 道路提取：per-patch oracle 的全域 F1 只从 0.3985 提到 0.4091，说明单纯调阈值帮助有限。
- 建筑物提取：patch 平均 F1 有提升，但全域 micro F1 没提升，说明不同 patch 的最佳阈值不稳定，直接用于生产会破坏整体精度。
- 因此当前主要瓶颈不是“统一阈值没选好”，而是概率图本身对目标/背景的排序还不够稳。
