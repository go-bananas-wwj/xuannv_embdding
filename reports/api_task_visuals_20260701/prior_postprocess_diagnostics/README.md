# 海淀道路/建筑地类先验后处理诊断

目的：缓解山体、绿地、水体等区域被误判为建筑/道路的问题。该版本不使用目标 GT 修改预测，只使用 WorldCover 与 OSM 水体/绿地作为外部先验做抑制。

|任务|Raw F1|Post F1|Raw Precision|Post Precision|Raw Recall|Post Recall|FP减少|FN增加|
|---|---:|---:|---:|---:|---:|---:|---:|---:|
|建筑物提取|0.4191|0.4225|0.3209|0.3286|0.6039|0.5916|34769|6160|
|道路提取|0.3985|0.4009|0.2932|0.3202|0.6220|0.5360|296489|70487|

结论：该后处理能减少部分山体/绿地/水体误报，精度略有提升，但召回会下降。它适合作为 API 的 `postprocessed` 图层，而不是替代原始概率图。下一步更稳的方案是加入 DEM 坡度、NDVI/纹理和训练期 hard negative。
