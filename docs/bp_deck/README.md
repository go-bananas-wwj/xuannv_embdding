# 玄女科技 BP 制作工作区

本目录集中保存商业 BP 制作相关的计划、素材、脚本和输出文件。

## 目录结构

- `plans/`：逐页内容计划、材料清单、叙事版本和来源说明。
- `assets/`：PPT 使用的图片、旧版 BP 可复用媒体、生成图和实验图裁片。
- `outputs/`：生成出来的 PPT 文件。
- `scripts/`：用于生成 PPT 的脚本。
- `user_materials/`：预留给用户补充 Logo、客户材料、团队照片、成本账本、三地样板数据等。
